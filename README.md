# OPKOD Claude Code Plugins

Claude Code plugin marketplace by OPKOD France.

## Setup

```bash
# 1. Add the marketplace
/plugin marketplace add opkod-france/opkod-claude-code-plugins

# 2. Install a plugin
/plugin install ui-polish@opkod-france
```
