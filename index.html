Curated plugins for Claude Code by OPKOD France. Enhance your AI-assisted development workflow.

Add This Marketplace
claude code
> /plugin marketplace add opkod-france/claude-plugins-marketplace

Run this command inside Claude Code to add the marketplace, then browse plugins with /plugin

Getting Started
01

Add Marketplace

Register this catalog with Claude Code

/plugin marketplace add opkod-france/claude-plugins-marketplace
02

Browse & Install

Open plugin manager, go to Discover tab

/plugin
03

Or Install Directly

Install a specific plugin by name

/plugin install ui-polish@opkod-france
Managing Plugins
—

List Installed

View all your installed plugins

/plugin → Installed tab
—

Disable / Enable

Toggle a plugin without removing

/plugin disable ui-polish@opkod-france
—

Uninstall

Completely remove a plugin

/plugin uninstall ui-polish@opkod-france
