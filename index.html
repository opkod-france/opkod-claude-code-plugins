Add Your Plugin

Want to contribute a plugin to this marketplace? Submit a pull request to add your plugin to our collection.

  1. Fork the marketplace repository
  2. Add your plugin entry to marketplace.json
  3. Create a plugin folder in plugins/your-plugin-name/
  4. Submit a pull request
